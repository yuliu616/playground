# client (vue-ws-hello)

## Development Setup

- npm (node.js v18)

```sh
npm install
```
## Debugging

```sh
# debug web will be served at: http://127.0.0.1:5173/
npm start
```

## Testing

- vitest
```sh
npm test
```

## Building

```sh
# build to output folder `dist`
npm run build
```

## Preview

- serving packaged:

```sh
npm run preview
```

- Entry points:

> http://localhost:4173/
